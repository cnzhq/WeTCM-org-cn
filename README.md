# 关于WeTCM
该网站基于Meta公司的开源项目[Docusaurus](https://docusaurus.io/)项目开发而来。用于分享个人的各种知识/内容，旨在于练习和实践。

这是一个**新手项目**。这是本人的第一次尝试网站开发。

## 实现原理
Markdown文本->GitHub Pull->GitHub Actions->中国大陆服务器部署
